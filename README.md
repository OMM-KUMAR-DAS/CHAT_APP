# CHAT_APP

This is a Very Basic chatting app where an user can create a particular room and chat  with a specific group of friends.
First you have to signup to this app and after which you can login and after this you will be redirected to page where you have to enter a meeting room in which you want to chat with your friends.
This app made using Websockets.

You can read the docs of websockets after clicking on the link provided below

https://socket.io/

Frameworks used

->Express.js
->Mongoose
->React and Redux Toolkit
->Web sockets
